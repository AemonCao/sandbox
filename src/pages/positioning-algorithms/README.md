# 多算法定位引擎对比系统

这是一个用于对比不同蓝牙定位算法性能的可视化工具，支持实时算法计算、性能分析和可视化展示。

## 🎯 功能特性

### 🔬 核心算法支持

- **三角定位算法 (Trilateration)** - 基于三个或更多信标的距离计算
- **指纹定位算法 (Fingerprinting)** - 基于RSSI指纹数据库匹配
- **质心算法 (Centroid)** - 简单的几何中心计算
- **加权质心算法 (Weighted Centroid)** - 基于信号强度的加权计算
- **卡尔曼滤波算法 (Kalman Filter)** - 递归状态估计和滤波
- **粒子滤波算法 (Particle Filter)** - 蒙特卡罗贝叶斯滤波

### 📊 可视化模式

- **叠加模式** - 同时显示真实位置和算法结果
- **对比模式** - 多算法结果并排对比
- **精度模式** - 误差分布热力图
- **热力图模式** - 信号强度覆盖热力图

### 🧪 测试场景

- **精度测试** - 网格化精度评估
- **密度测试** - 不同信标密度的影响
- **噪声鲁棒性测试** - 各种噪声环境下的表现
- **可扩展性测试** - 大规模环境性能测试
- **边界情况测试** - 边缘和遮挡场景
- **动态测试** - 移动目标追踪性能
- **算法对比测试** - 全面的性能指标对比

## 🎮 使用方法

### 1. 快速开始

1. 访问 `http://localhost:3333/positioning-algorithms`
2. 系统会自动加载默认场景（4个信标，10个测试点）
3. 点击"重新计算"按钮开始算法对比
4. 在左侧画布查看可视化结果
5. 在右侧面板查看详细分析

### 2. 配置算法参数

- 在"算法配置"面板中展开具体算法
- 调整算法参数（如K值、噪声阈值等）
- 实时查看参数变化对结果的影响

### 3. 运行测试场景

- 在"测试控制"面板选择测试场景
- 点击"开始测试"运行自动化测试
- 查看测试进度和结果统计

### 4. 导出结果

- 点击"生成报告"导出Markdown格式的详细报告
- 支持JSON格式的原始数据导出
- 自动保存配置到本地存储

### 算法实现特点

- **三角定位**: 最小二乘法求解圆交点
- **指纹定位**: K近邻 + 加权插值
- **卡尔曼滤波**: 状态空间模型递推估计
- **粒子滤波**: 蒙特卡罗采样和重采样

- **三角定位**: 权重系数、噪声滤波、异常值阈值
